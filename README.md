# Portfolio

A portfolio website showcasing projects and information.

## Description

This is a web-based portfolio built with HTML, CSS, and JavaScript. It displays various pages including a main portfolio page and a dedicated pets section.

## Features

- Responsive design with CSS and SASS styling
- Multiple pages for different content sections
- Interactive elements powered by JavaScript
- Organized asset management with images and stylesheets

## Project Structure

```
portafolio/
├── index.html          # Main portfolio page
├── pets.html          # Pets page
├── script.js          # JavaScript functionality
├── img/               # Image assets
└── assets/
    ├── css/           # Compiled CSS stylesheets
    │   ├── index.css
    │   ├── pets.css
    │   └── styles.css
    ├── img/           # Additional images
    └── sass/          # SASS source files
        ├── index.scss
        ├── pets.scss
        └── styles.scss
```

## Getting Started

1. Clone from https://github.com/Lmedrano21/portafolio.git
2. Open `index.html` in your web browser to view the portfolio
4. Navigate between pages using the links in the site

## Technologies Used

- HTML5
- CSS3 / SASS
- JavaScript

## Notes

The project uses SASS for styling organization. Make sure to compile SASS files to CSS if you make changes to `.scss` files.
